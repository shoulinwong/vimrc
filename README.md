# Key Mappings

The leader is `,`.

## Plugin related mapping

- `<c-j><c-n>` :Open Nerdtree
- `<c-j><c-t>` :Open Tagbar
- `<c-b>` :Open FZF buffers
- `<c-p>` :Open FZF files
- `<leader>g` :Ack Search using ripgrep
